# Geometry-processing-and-deep-learning

---

## Description of the work :

---

### TD 1 :

- basic computation and data manipulation on meshes (computation of Laplacian, cotan matrix ...)

- exact diffusion on surface using Laplace-Beltrami operator

- approximate diffusion on surface by solving  the generalized eigen value problema

- computation of the Heat Kernel Signature (HKS) descriptor  to compare surface

- Heat Method for geodesic computation

---

### TD 2 :

- Tutte embedding

- Least Square Conformal Map (embedding that preserve angles and circles)

- Surface Alignement: Iterative Closest Point (ICP) implementation

- As-Rigid-As-Possible (ARAP)  implementation

- Non-Rigid ICP implementation

---

### TD 3 (incoming) :

- PointNet implementation

---

### TD 4 :

- Implementation of DiffusionNet

- RNA segmentation using DiffusionNet

- Ablation studies of DiffusionNet

---

### TD 6 :

- SIREN Implementation

- Implicit Shape Interpolation
